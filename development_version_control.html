This semester I went on a journey to solidify my understanding of core JavaScript concepts. Through workshops and self-study, I honed my skills in areas like: rules and variables, scope, operation, data types, functions, objects, events, arrays, if and switch statements, comparisons, logical operators and different types of loops.

Additionally, I learned VueJS and some React and passed an interview for a Full-Stack developer at a company in Eindhoven (More about that in Learning Outcome 5).

I also looked for and applied good practices and common practices that people in big tech companies use like making everything semantic (instead of just using divs), using naming conventions, making everything smoothly responsive, keeping the code clean and clear, so it’s readable and people after can easily follow up.

I learned how to make a smooth dark theme change with DOM manipulation. See it yourself by clicking the dark theme in my portfolio website.

Bringing Concepts to Life

To solidify my grasp of JavaScript, I actively participated in workshops and applied my knowledge to create engaging projects:

  • Tic-Tac-Toe with a Twist: I built a classic tic-tac-toe game, but with a fun twist! Upon winning, the successful combination lights up in green and a celebratory Pikachu animation dances across the screen. This project demonstrates my understanding of game logic, conditional statements (if statements), and DOM manipulation (changing box colors and displaying animations).
  • Pokemon Pixel Art: I created a pixelated rendition of a Pokemon, showcasing my ability to translate visual elements into code and understanding of grid systems.
  • Real-Time Chat Application (Advanced): This project delves deeper into server-side JavaScript (Node.js) for real-time interaction and utilizes Socket.io for communication, allowing users to chat in real-time.
  • Pokemon Fighting Game using an API (Advanced): Taking my skills a step further, I built a Pokemon fighting game by fetching data from an external API and implementing the mechanics using JavaScript.
  • AR Pikachu: Creating and running an Augmented Reality Server Application, displaying a 3D Model in real life through a smartphone camera.
  • Github Web Accessibility Plan: I ran an evaluation tool analysis and made a plan to make the GitHub website more optimized for visually impaired people.
